## 论文点评

### Agentic World Modeling
- **Title**: Agentic World Modeling: Foundations, Capabilities, Laws, and Beyond
- **Institutes**: 未明确标注（40+ 作者）
- **Source**: [link](https://arxiv.org/abs/2604.22748) 🔥 HF Trending ⬆️ 181  **📒 论文笔记**: [[2604-AgenticWorldModel]]
- **核心**: Survey 提出世界模型的双轴分类：能力层级（L1 Predictor → L2 Simulator → L3 Evolver）× 约束域（Physical/Digital/Social/Scientific），综合 400+ 工作，连接碎片化社区。
- **锐评**: Taxonomy 设计精巧，正交且覆盖核心差异。"levels x laws" 可能成为世界模型研究的标准框架。但 40+ 作者的 coordination overhead 和 L3 Evolver 的可行性存疑。Survey 类工作，需要看全文判断 depth。
- **Rating**: `3` 🔥 — Taxonomy 设计有 insight，可能成为领域标准。

### SkillClaw
- **Title**: SkillClaw: Let Skills Evolve Collectively with Agentic Evolver
- **Institutes**: 未标注
- **Source**: [link](https://arxiv.org/abs/2604.08377) 🔥 HF Trending ⬆️ 286  **📒 论文笔记**: [[2604-SkillClaw]]
- **核心**: Agent skill 的集体演化：跨用户轨迹聚合 → autonomous evolver 提取 pattern → 更新共享 skill 库 → 同步所有用户。核心洞察：跨用户交互是 skill 进化的 primary signal。
- **锐评**: 286 upvotes 热度极高，核心 idea 简洁清晰。但 abstract 缺少具体数字（"显著提升"无量化），evolver 的设计细节未知（如何识别 good vs bad pattern？），跨用户 skill 共享的隐私问题未提及。
- **Rating**: `3` 🔥 — Idea 清晰，热度高，但需要看全文验证 evolver 设计。

### LLaDA2.0-Uni
- **Title**: LLaDA2.0-Uni: Unifying Multimodal Understanding and Generation with Diffusion Large Language Model
- **Institutes**: Inclusion AI
- **Source**: [link](https://arxiv.org/abs/2604.20796) 🔥 HF Trending ⬆️ 232  **📒 论文笔记**: [[2604-LLaDA2Uni]]
- **核心**: 离散 diffusion LLM，原生统一多模态理解+生成。架构：SigLIP-VQ tokenizer → MoE dLLM backbone → diffusion decoder。开源。
- **锐评**: 232 upvotes 热度极高。真正的 unified model（不是两模块拼接），discrete diffusion 思路不同于主流 continuous diffusion。但 abstract 缺少具体 benchmark 数字，MoE + diffusion 的显存和推理速度可能有问题。
- **Rating**: `3` 🔥 — Unified 多模态有 insight，热度高，开源诚意足。
