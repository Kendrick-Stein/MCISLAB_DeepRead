## 论文点评

### World2VLM
- **Title**: World2VLM: Distilling World Model Imagination into VLMs for Dynamic Spatial Reasoning
- **Institutes**: Not specified
- **Source**: [link](http://arxiv.org/abs/2604.26934v1)  📄 arXiv   **📒 论文笔记**: [[Papers/2604-World2VLM]]
- **核心**: 提出训练时蒸馏框架，将 generative world model 的空间想象能力注入 VLM，而非推理时耦合 WM。给定初始观测 + camera trajectory，用 view-consistent WM 合成几何对齐的未来视角，构造 forward（action→outcome）和 inverse（outcome→action）空间推理的结构化监督。两阶段 post-training 在 SAT-Real/SAT-Syn/VSI-Bench/MindCube 上一致超越 base model，且超越 test-time WM-coupled 方法。
- **锐评**: 核心 idea 清晰——"WM as training-time teacher rather than inference-time tool"——且实验设计干净（4 个 benchmark 一致提升 + test-time 方法对比）。但 dataset 规模和 WM 质量的 scaling behavior 未讨论，compact dataset 的"compact"程度未量化。WM 本身的 geometric alignment 质量上限决定了蒸馏天花板，若 WM 有 hallucination 则会 transfer 到 VLM。
- **Rating**: `2` 👀 — 训练时蒸馏 vs 推理时耦合的 trade-off 分析有价值，但 WM 质量瓶颈未解

### STARRY
- **Title**: STARRY: Spatial-Temporal Action-Centric World Modeling for Robotic Manipulation
- **Institutes**: Not specified
- **Source**: [link](http://arxiv.org/abs/2604.26848v1)  📄 arXiv   **📒 论文笔记**: [[Papers/2604-STARRY]]
- **核心**: 提出 action-centric 空间-时序世界模型，核心 insight 是 world model 应优化 control utility 而非 visual realism。Joint denoising future spatial-temporal latents + action sequences，GASAM 将预测的 depth 和 end-effector geometry 转换为 token-aligned weights 做 selective action-attention modulation。RoboTwin 2.0 达 93.82%/93.30%（Clean/Randomized），real-world 从 42.5% 提升到 70.8% over π0.5。
- **锐评**: 93.8% 在 RoboTwin 2.0 上是 impressive 的数字，GASAM 的设计（3D geometry → attention bias）简洁且有物理直觉支撑。但 real-world 只有 3 个任务、1 个平台，70.8% 的置信区间很宽。计算 overhead 未讨论——joint denoising latents + action 的推理成本是多少？与 π0.5 的对比是否公平（相同 compute budget？）。
- **Rating**: `2` 👀 — "WM for control not realism" 的 insight 有价值，但 real-world 验证太窄

### Beyond Screenshots
- **Title**: Beyond Screenshots: Evaluating VLMs' Understanding of UI Animations
- **Institutes**: Not specified
- **Source**: [link](http://arxiv.org/abs/2604.26148v1)  📄 arXiv   **📒 论文笔记**: [[Papers/2604-BeyondScreenshots]]
- **核心**: 首个 UI 动画理解 benchmark UIBench，覆盖 7 类问题（从基础感知到因果推理）。评测 15 个 VLM，发现 top models 显著落后人类，多数在推理任务上接近随机。核心发现是 perception-reasoning gap：模型能描述动画外观但不能解释为什么发生或对用户交互意味着什么。提出 multi-view prompting 和 animation-to-description 两种缓解策略。
- **锐评**: 问题定义好——UI 动画是 GUI Agent 必须理解但被忽视的 modality。perception-reasoning gap 的发现有普遍意义（不只限于 UI）。但 benchmark 规模和多样性未在摘要中透露，7 类问题的覆盖度存疑。两种缓解策略的效果未量化。
- **Rating**: `2` 👀 — 问题定义有价值（UI 动画理解被忽视），perception-reasoning gap 发现有普遍意义
